Portfolio – Reflectie WPL1

Situering binnen Profiel DVO

Waar sta ik nu?

Ik sta aan het begin van mijn ontwikkeling als programmeur/designer. Ik begrijp de basis van HTML, CSS en JavaScript en kan eenvoudige websites structureren en opbouwen.

Wat kan ik al?

Ik kan een website opbouwen met een duidelijke structuur, gebruik maken van semantische HTML en eenvoudige interactiviteit toevoegen met JavaScript.Ook kan ik een aanleg maken naar een mooi basis design voor een website.

Wat wil ik nog verbeteren?

Ik wil vooral mijn skills in talen zoals javascript, C# verder ontwikkelen, verder wil ik ook mijn grafische skills verder ontwikkelen om zo meer profesionelen ontwerpen te creeren

Eindreflectie WPL1

Wat ging goed?

Het zelfstandig werken aan mijn portfolio ging goed. Ik heb geleerd om mijn code beter te organiseren en stap voor stap te werken.

Wat was moeilijk?

Het structureren van grotere projecten en het correct gebruiken van GitHub was in het begin moeilijk. Soms liep ik vast door kleine fouten in mijn code.

Ook het werken in groep was voor mij niet zo eenvoudig omdat ik het liefst alleen werk, en de samenwerking ging niet altijd even goed.

Wat heb ik geleerd?

Ik heb geleerd dat fouten maken een belangrijk deel is van het leerproces. Door te blijven testen en opzoeken begrijp ik mijn werken beter

Ook het opvolgen van deadlines en het plannen van mijn werk zijn vaardigheden die ik heb verbeterd.
